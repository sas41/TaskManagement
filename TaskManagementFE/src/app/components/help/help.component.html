Welcome to the TaskMan Demo!

By default, you won't be logged in, so the first thing to do is log in!

Use the following credentials:

Username Password Role
admin admin Admin
man1 man1 Manager
man2 man2 Manager
user1 user1 Standard User
user2 user2 Standard User

Admins can do just about everything in the app, except edit other people's comments.

Managers can create new tasks and edit or delete their own tasks, as well as assign anyone to their tasks.

Everyone can post and edit their own comments as well as view tasks or do a general search.

Clicking on a task on the task list will show you that task.

Clicking on a comment or task in the search page will also open the associated task.

When adding a comment, it can be set as a reminder, for which a date must be picked.
